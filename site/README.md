# Ariadne project site

The public site for [Ariadne](https://github.com/mclaut/ariadne), built with
Next.js-compatible Vinext and deployed with OpenAI Sites.

```bash
npm ci
npm run dev
```

Production validation:

```bash
npm run lint
npm test
```

The generated release social card is `public/og.png`; its source visual is kept
in `public/og-visual.png`. Search and AI discovery files live in `public/`.
