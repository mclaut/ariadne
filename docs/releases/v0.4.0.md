## What's new

- **Estimated token savings.** Ariadne now measures the context delivered by
  automatic and MCP recalls and the original session context represented by
  newly captured diary memories. Net avoided context is reported as
  `represented - delivered` with a visible `~` marker.
- **Local metrics CLI and JSON.** `ariadnectl metrics` shows all-time and rolling
  30-day totals. `ariadnectl metrics -json` provides stable fields for recalls,
  memories, delivered tokens, represented tokens, and net avoided tokens.
  `ariadnectl status -json` exposes the same summary to monitors.
- **Tray visibility in seven languages.** The green tray menu now shows the
  current all-time token-savings estimate alongside health, record count, and
  disk usage on Windows, macOS, and Linux.
- **Content-free by design.** The metrics database stores numeric counters and
  opaque event hashes only, never memory or transcript text. It lives at
  `~/.ariadne/metrics.db` with user-only permissions and can be disabled for new
  events with `ARIADNE_METRICS=0`.
- **Conservative accounting.** Existing memories without source-size metadata
  receive no invented savings. Repeated automatic delivery still counts as
  overhead while represented context is credited once per session.

## How the estimate works

Ariadne uses a deterministic, multilingual UTF-8 byte estimate because exact
tokenization depends on the MCP client and model. The result measures context
reuse, not provider billing. New curated session captures store only their
numeric source and memory sizes in Qdrant payload metadata; the source text is
handled exactly as before and raw transcripts are never stored.

```bash
ariadnectl metrics
ariadnectl metrics -json
```

No database migration or re-embedding is required. Counters begin accumulating
after upgrading, and savings become available as newly captured diary memories
are recalled.

## Install or update

### Windows

```powershell
irm https://raw.githubusercontent.com/mclaut/ariadne/main/install.ps1 -OutFile install.ps1
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

### Linux or macOS

```bash
curl -fsSL https://raw.githubusercontent.com/mclaut/ariadne/main/install.sh | sh
```

The tray also detects `v0.4.0` and offers the same consent-gated update flow.
